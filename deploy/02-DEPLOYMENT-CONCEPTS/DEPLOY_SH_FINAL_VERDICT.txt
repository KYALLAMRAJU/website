IMPROVEMENT #1: Secure PostgreSQL Password Handling
────────────────────────────────────────────────────
Before:
  CREATE USER advaitam_user WITH PASSWORD 'CHANGE_THIS_PASSWORD';

After:
  DB_PASSWORD="${DB_PASSWORD:-CHANGE_THIS_PASSWORD_IMMEDIATELY}"
  if [ "$DB_PASSWORD" = "CHANGE_THIS_PASSWORD_IMMEDIATELY" ]; then
      echo "⚠️  WARNING: Using default database password!"
      echo "You MUST set DB_PASSWORD environment variable..."
  fi
  CREATE USER advaitam_user WITH PASSWORD '$DB_PASSWORD';

Benefits:
  ✅ Can pass password via environment variable
  ✅ Warns user if using default password
  ✅ Prevents accidental weak password in production

Usage:
  sudo DB_PASSWORD='your-strong-password' bash deploy.sh

After:
